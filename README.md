# Authentication system using Reactjs and Firebase - [Link](https://fir-login-methods.firebaseapp.com/)

- I created full authentication system using reactjs and firebase with some awesome styling.
- For styling I used css3 with preprocessor sass, and for icons I used Font Awesome icons.
- I used reusable component for button.

## Within this app you can login with:
### Facebook - [Link](https://developers.facebook.com/)
### Google
### Github - [Link](https://github.com)
- Go to your github profile.
- Open settings and look for Developer settings -> OAuth Apps -> Create new OAuth app
